::uv_file uv::file::fd ( ) const
inlinenoexcept

Get the cross platform representation of the file handle.

On Windows this function returns a C run-time file descriptor which differs from the operating-system file handle that is returned by handle::fileno() function. On Unix-like systems both functions return the same value.

See also
Windows: _open_osfhandle().
