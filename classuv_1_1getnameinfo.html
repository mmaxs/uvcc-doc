Run the request.

See also
libuv API documentation: uv_getnameinfo().
Note
If the request callback is empty (has not been set), the request runs synchronously.

Available _NI_FLAGS are:

  • NI_DGRAM
  • NI_NAMEREQD
  • NI_NOFQDN
  • NI_NUMERICHOST
  • NI_NUMERICSERV
