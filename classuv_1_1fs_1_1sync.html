See also
libuv API documentation: uv_fs_fsync(), uv_fs_fdatasync().
Linux: fsync(), fdatasync().

By default uv_fs_fdatasync() libuv API function is used.

Note
If the request callback is empty (has not been set), the request runs synchronously.
