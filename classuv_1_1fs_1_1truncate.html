Run the request. Truncate the _file to a size of _offset bytes.

See also
libuv API documentation: uv_fs_ftruncate().
Linux: ftruncate().
Note
If the request callback is empty (has not been set), the request runs synchronously.

The _offset value of < 0 means using of the current file position.

See also
Windows: _tell(), _telli64().
